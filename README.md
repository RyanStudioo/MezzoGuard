# Mezzo Guard

Mezzo Guard is a Python SDK designed to add a protective layer to LLM applications attacks such as jailbreaking and prompt injection, powered by custom-trained models adapted to recent trends.

Mezzo Guard can be easily added into your pipeline, offering great protection with little effort.

## Supported Models
### Prompt Guards
